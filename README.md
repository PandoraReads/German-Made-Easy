# German Made Easy (GME) - Obsidian German Learning Assistant

**German Made Easy (GME)** is an immersive Obsidian learning plugin designed specifically for German language learners. It combines instant word lookup, spaced repetition (FSRS) flashcard review, and video shadowing practice to help you build a seamless learning experience and German corpus through one-stop learning, mastering the language in the most efficient way possible.

## ✨ Core Features

### 1. 🔍 Immersive Word Assistant (Dictionary Assistant)
*   **Quick Lookup**: Support `Ctrl/Cmd + Double-click` or `Drag to select` to instantly display definitions in the sidebar.
*   **Contextual Precision**: Automatically captures complete sentences containing the word as example sentences during lookup, eliminating fragmented memory.
*   **Offline Dictionary Support**: Built-in Youdao/Google/Bing online services, with support for loading local **MDX/MDD** professional dictionaries.
*   **Data Persistence**: One-click add to vocabulary notebook, with data automatically synced to IndexedDB and local `生词本.md` file.

### 2. 🎨 Highlight Note Cards (Highlight Note Cards)
*   **Sidebar Management**: Aggregates `<mark>` highlighted content from current documents or all vault files, supporting **instant navigation** to original text with one click.
*   **Multi-dimensional Filtering**:
    *   **Color Filtering**: Quick categorization by four highlight colors: yellow, pink, blue, and green.
    *   **Keyword Search**: Real-time filtering of note content via top search bar.
*   **AI Deep Empowerment**:
    *   **AI Translation (Languages)**: One-click access to precise translations of highlighted sentences.
    *   **AI Research (Microscope)**: **Three-layer philosophical deep analysis** (essence extraction, conceptual perspective, practical path) providing philosophical background expansion.
*   **Flexible Interaction**:
    *   **Quick Editing**: Click pencil icon to directly edit notes on cards with auto-adjusting height.
    *   **Tag System**: Support custom tags for cards with built-in intelligent recommendation algorithm.
    *   **Card Collection Navigation**: One-click switch to fullscreen "Card Studio" mode for centralized inspiration management.

### 3. 🧠 FSRS Flashcard Review System (Flashcard System)
*   **Scientific Algorithm**: Integrated state-of-the-art **FSRS (Free Spaced Repetition Scheduler)** scheduling algorithm.
*   **Visual Excellence**: Stunning **Glassmorphism** UI design with 3D card flip interactions.
*   **Learning Dashboard**: Built-in learning heatmap and statistics dashboard, making progress clearly visible.

### 4. 🎬 Video Shadowing Workshop (Shadowing Workshop)
*   **Multi-mode Learning**: One-click toggle between **Shadowing Mode** and **Dictation Mode**.
*   **Precise Playback Control**: Specially designed control bar:
    *   **Speed Adjustment**: Flexible switching between **0.8x / 1.0x / 1.25x**.
    *   **Quick Positioning**: **-5s / -10s** quick jump back for repeated pronunciation practice.
*   **Full Format Support**: Perfect support for YouTube and local videos with real-time subtitle synchronization and auto-scroll during playback.

## 🚀 Quick Start

### Installation
1.  **Manual Installation (Recommended):**
    *   Obtain the latest version's three main files: `main.js`, `manifest.json`, `styles.css`.
    *   Place them in your vault folder: `.obsidian/plugins/obsidian-germany-made-easy/`.
    *   Enable the plugin in Settings → Third-party plugins.
2.  **Community Plugins (Coming Soon):** Search for `Germany Made Easy` in Obsidian's community plugin market and install.

### Basic Configuration
1.  **Dictionary Settings**: Choose your preferred dictionary source in settings (Youdao or local MDX recommended).
2.  **Vocabulary Path**: Specify a folder for the auto-generated `生词本.md`.

## 🌐 Language Support

German Made Easy now supports multiple interface languages:

*   **中文**: Full Chinese localization for native Chinese speakers learning German
*   **English**: Complete English interface for native English speakers learning German

Switch the interface language in plugin settings at any time. The plugin automatically adjusts:
*   Interface language (menus, buttons, prompts)
*   Dictionary direction (German-Chinese ↔ German-English)
*   AI translation prompts (outputs result in your chosen language)
